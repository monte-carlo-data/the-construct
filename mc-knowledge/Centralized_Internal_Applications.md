# Centralized Internal Applications

This file lists approved internal applications and SaaS tools. Used by Oracle when
evaluating whether a detected application or integration is authorized.

---

## How to use

Add a row for each approved tool. Oracle checks detected apps against this list
and flags anything not present as potential shadow IT.
